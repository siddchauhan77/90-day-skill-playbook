## Shaan's Framework: Which Categories This Hits

| Category | How Web Dev Fits |
|----------|-----------------|
| **Making** (primary) | You literally create products — websites, apps, tools — that didn't exist before |
| **Designing** (secondary) | Good web dev requires taste: layout, UX, visual hierarchy, what feels right |

This is the purest **Making** skill on the list. You go from idea to working product. That's a superpower in a world where most people can only talk about ideas.

---

## No-Code vs Full-Stack: Which Path?

| Path | Tools | Time to Proficiency | Ceiling |
|------|-------|-------------------|---------|
| **No-Code** | Webflow, Framer, Bubble | 60–90 days | $5k–$15k/project |
| **Full-Stack** | HTML/CSS/JS + React/Node | 120–180 days | $150k+/year employed, unlimited freelance |

**Recommendation for most people:** Start no-code (Webflow or Framer). You'll be building real sites in weeks. If you love it, go full-stack after.

## Shaan's Rule 2: How Web Dev Becomes Equity

| Vehicle | What It Looks Like |
|---------|--------------------|
| **Code** (primary) | Build a Webflow template store, SaaS tool, or productized web agency — one asset, infinite customers |
| **Content** | Document your builds on Twitter/YouTube (massive developer audience) |
| **Capital** | Build your own micro-SaaS and own equity in it |

The Shopify themes example from Shaan's playbook is web dev. Designer quit his $200k job, made one theme, sold it to 2,000 customers while sleeping. **$2M/year.** Same skill — different business model.

## What $10k/Project Web Developers Do Differently

1. They sell outcomes, not hours ("your site will convert 2x more leads")
2. They understand the client's business, not just the design brief
3. They deliver fast (speed is a massive differentiator)
4. They package services (design + copy + launch = higher price)
5. They build a portfolio that shows results, not just aesthetics
